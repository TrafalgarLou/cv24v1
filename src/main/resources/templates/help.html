Liste des routes de l'application CV24 :

/cv24/html

Méthode : GET

Résumé : Affiche les détails d'un CV en format HTML.

Retour : HTML

/cv24/xml

Méthode : GET

Résumé : Retourne les détails d'un CV en format XML.

Retour : XML

/cv24/delete/{id}

Méthode : DELETE

Résumé : Supprime un CV de la base de données.

Retour : JSON (status de l'opération)

/cv/resume/html

Méthode : GET

Résumé : Affiche un résumé des CV en format HTML.

Retour : HTML

/validate-xml

Méthode : POST

Résumé : Valide un document XML par rapport à un schéma XSD.

Retour : Texte (résultat de la validation)
